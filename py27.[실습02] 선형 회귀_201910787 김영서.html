3. Gradient descent -> theta¶

  • 선형회귀모델 #### $h_{\theta}(x) = {\theta}_0 + {\theta}_1x$
  • 경사 하강법 알고리즘

    for j=0 and j=1 {
    $ {\theta}_j = {\theta}_j - {\alpha}\frac{\partial}{\partial{\theta}_j}J({\theta}_0, {\theta}_1) $
    }

  • 그라디언트
    $\frac {δ} {δθ_{j}}J(θ_{0}, θ_{1}) = \left(\begin{array}{r} \frac {δ} {δθ_{0}}J(θ_{0}, θ_{1})\\ \frac {δ} {δθ_{1}}J(θ_{0}, θ_{1})\end{array}\right) = \left(\begin{array}{r} \frac {1} {m}\Sigma_{i=1}^{m}(θ_{0} + θ_{1}x^{(i)} - y^{(i)})·1\\ \frac {1} {m}\Sigma_{i=1}^{m}(θ_{0} + θ_{1}x^{(i)} - y^{(i)})·x^{(i)}\end{array}\right) = \frac {1} {m} X^{T}_{b}·(X_{b}·θ - y)$

X_b = np.c_[np.ones((m,1)), X] # 모든 샘플에 x0=1을 추가
# c_ : concatenation. 배열을 옆으로 붙이기

learning_rate = 0.0001 # 학습률(learning rate)
n_iter = 200

theta = np.zeros((2,1))
gradients = np.zeros((2,1))

for i in range(n_iter):
    gradients = (np.dot((X_b.T), (np.dot(X_b,theta) - y)))/m
    theta = theta - learning_rate * gradients

def ComputeJ(t0, t1, X, y):
    m = len(X)
    X_b = np.c_[np.ones((m,1)), X] # 모든 샘플에 x0=1을 추가
    theta = np.array([t0, t1])
    
    cost_vector = X_b.dot(theta) - y # (52, 1)
    
    cost_vector = cost_vector.reshape(-1)
    # 각 원소 제곱
    cost2 = np.square(cost_vector)
    # 원소들 더하기
    cost_sum = np.sum(cost2) / (2*m)
    
    return cost_sum

# 1. theta0, theta1, J 값 구하기
d = 100 # # of split

s = 1.0
theta0 = np.linspace(-15*s, 15*s, d) # -0.1 ~ 0.1
theta1 = np.linspace(-1.0*s, 1.0*s, d) # 0.1 ~ 0.2

J = np.ones((d, d)) # (100, 100)
for i,t0 in enumerate(theta0):
    for j, t1 in enumerate(theta1):
        cost = ComputeJ(t0, t1, X, y)
        J[i, j] = cost
